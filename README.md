# Python-Math-quiz-generator
A Python-based CLI application that generates random multiplication tables for practice. Features customizable table selection, adjustable question counts, and real-time scoring.

Multiplication Table Trainer is a legacy Python project originally developed during my 8th-grade year. It serves as a personal milestone in my journey as a programmer. The script provides an interactive Command Line Interface (CLI) where users can specify a multiplication table and the number of rounds they wish to practice.
By utilizing the random module for question generation and the time module to pace the user experience, this project highlights early implementations of logic-based learning tools and basic state management in Python.
